### 230. Kth Smallest Element in a BST

给定一个二叉搜索树的根节点 `root` ，和一个整数 `k` ，请你设计一个算法查找其中第 `k` 个最小元素（从 1 开始计数）。

 **示例 1：**

![kthtree1](E:\leetcode\230.Kth Smallest Element in a BST\kthtree1.jpg)

```
输入：root = [3,1,4,null,2], k = 1
输出：1
```
